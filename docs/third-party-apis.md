# 第三方股票数据接口文档

本文档整理了 [`stock-api`](https://github.com/zhangxiangliang/stock-api.git) 项目使用的第三方股票数据接口详情，包括腾讯、新浪、东方财富三大数据源。


---

## 一、腾讯股票接口

### 1.1 获取股票行情

**接口地址**  
```
GET https://qt.gtimg.cn/q={codes}
```

**请求参数**

| 参数名 | 类型   | 必填 | 说明                              |
|--------|--------|------|-----------------------------------|
| codes  | string | 是   | 股票代码，多个代码用逗号分隔      |

**股票代码格式**
- 深交所：`sz` + 代码（如：`sz000651`）
- 上交所：`sh` + 代码（如：`sh510500`）
- 港交所：`hk` + 代码（如：`hk02020`）
- 美股：`us` + 代码（如：`usdji`）

**请求示例**
```
https://qt.gtimg.cn/q=sh510500,sz000651
```

**响应格式**

编码：GBK  
分隔符：`~`  
返回格式：`v_{code}="{data}"`

**响应示例**
```
v_sh510500="1~500ETF~510500~7.224~7.149~7.147~2724781~1351032~1373749~7.224~63~7.223~329~7.222~370~7.221~286~7.220~143~7.225~3893~7.226~884~7.227~323~7.228~5728~7.229~190~~20200731154041~0.075~1.05~7.280~7.085~7.224/2724781/1964297124~2724781~196430~~~~7.280~7.085~2.73~~~0.00~7.864~6.434~1.20~-9827~7.209~~~~~~196429.7124~0.0000~0~ ~ETF~26.51~~~~";
```

**字段说明（数组索引）**

| 索引 | 字段     | 说明         | 示例   |
|------|----------|--------------|--------|
| 1    | name     | 股票名称     | 500ETF |
| 3    | now      | 当前价格     | 7.224  |
| 4    | yesterday| 昨收价格     | 7.149  |
| 33   | high     | 最高价       | 7.280  |
| 34   | low      | 最低价       | 7.085  |

**注意事项**
- 接口返回编码为 GBK，需转换为 UTF-8 处理
- 涨跌幅需计算：`percent = now / yesterday - 1`
- 批量查询时，未找到的股票不会包含在响应中

### 1.2 搜索股票

**接口地址**
```
GET https://smartbox.gtimg.cn/s3/?v=2&t=all&c=1&q={keyword}
```

**请求参数**

| 参数名  | 类型   | 必填 | 说明           |
|---------|--------|------|----------------|
| v       | string | 是   | 版本号，固定值 `2` |
| t       | string | 是   | 类型，固定值 `all` |
| c       | string | 是   | 参数，固定值 `1` |
| q       | string | 是   | 搜索关键词     |

**请求示例**
```
https://smartbox.gtimg.cn/s3/?v=2&t=all&c=1&q=格力电器
```

**响应格式**

编码：GBK  
返回格式：`v_hint="{type1}~{code1}^{type2}~{code2}..."`

**响应示例**
```
v_hint="sz~000651^sh~510500^hk~02020"
```

**类型映射**
- `sz` → 深交所股票
- `sh` → 上交所股票
- `hk` → 港交所股票
- `us` → 美股

## 二、新浪股票接口

### 2.1 获取股票行情

**接口地址**
```
GET https://hq.sinajs.cn/list={codes}
```

**请求头**
```
Referer: https://finance.sina.com.cn/
```

**请求参数**

| 参数名 | 类型   | 必填 | 说明                          |
|--------|--------|------|-------------------------------|
| codes  | string | 是   | 股票代码，多个代码用逗号分隔  |

**股票代码格式**
- 深交所：`sz` + 代码（如：`sz000651`）
- 上交所：`sh` + 代码（如：`sh510500`）
- 港交所：`hk` + 代码（如：`hk02020`）
- 美股：`gb_` + 代码（如：`gb_dji`）

**请求示例**
```
https://hq.sinajs.cn/list=sh510500,sz000651
```

**响应格式**

编码：GB18030  
分隔符：`,`  
返回格式：`var hq_str_{code}="{data}"`

**响应示例**
```
var hq_str_sh510500="中证500ETF南方,8.375,8.308,0.000,0.000,0.000,8.308,8.308,0,0.000,34400,8.308,100,0.000,0,0.000,0,0.000,34400,8.308,0,0.000,0,0.000,0,0.000,0,0.000,0,0.000,2026-04-24,09:16:15,00,";
```

**字段说明（因市场而异）**

**A股（深交所/上交所）字段位置**
| 索引 | 字段     | 说明         |
|------|----------|--------------|
| 0    | name     | 股票名称     |
| 2    | yesterday| 昨收价格     |
| 3    | now      | 当前价格     |
| 4    | high     | 最高价       |
| 5    | low      | 最低价       |

**港股字段位置**
| 索引 | 字段     | 说明         |
|------|----------|--------------|
| 1    | name     | 股票名称     |
| 3    | yesterday| 昨收价格     |
| 4    | high     | 最高价       |
| 5    | low      | 最低价       |
| 6    | now      | 当前价格     |

**美股字段位置**
| 索引 | 字段     | 说明         |
|------|----------|--------------|
| 0    | name     | 股票名称     |
| 1    | now      | 当前价格     |
| 6    | high     | 最高价       |
| 7    | low      | 最低价       |
| 26   | yesterday| 昨收价格     |

**注意事项**
- 必须携带 `Referer` 请求头，否则接口会拒绝请求
- 接口返回编码为 GB18030
- 涨跌幅需计算：`percent = now / yesterday - 1`
- 未找到的股票返回值为空字符串：`var hq_str_xxxx="";`

### 2.2 搜索股票

**接口地址**
```
GET http://suggest3.sinajs.cn/suggest/type=2&key={keyword}
```

**请求头**
```
Referer: https://finance.sina.com.cn/
```

**请求参数**

| 参数名 | 类型   | 必填 | 说明       |
|--------|--------|------|------------|
| type   | string | 是   | 类型，固定值 `2` |
| key    | string | 是   | 搜索关键词 |

**请求示例**
```
http://suggest3.sinajs.cn/suggest/type=2&key=格力电器
```

**响应格式**

编码：GB18030  
返回格式：`var suggestvalue="{code1},{name1};{code2},{name2};..."`

**响应示例**
```
var suggestvalue="sz000651,格力电器;sh510500,中证500ETF";
```

**代码前缀说明**
- `sz` - 深交所股票
- `sh` - 上交所股票
- `hk` - 港交所股票
- `us` - 美股
- `of` - 基金（需同时尝试深交所和上交所）

## 三、东方财富股票接口

### 3.1 获取股票行情

**接口地址**
```
GET https://push2.eastmoney.com/api/qt/ulist.np/get?fltt=2&secids={codes}&fields={fields}
```

**请求头**
```
Referer: https://quote.eastmoney.com/
Accept: application/json,text/plain,*/*
```

**请求参数**

| 参数名  | 类型   | 必填 | 说明                                    |
|---------|--------|------|-----------------------------------------|
| fltt    | string | 是   | 固定值 `2`                              |
| secids  | string | 是   | 股票代码，多个代码用逗号分隔            |
| fields  | string | 是   | 返回字段列表                            |

**股票代码格式**
- 上交所：`1.` + 代码（如：`1.510500`）
- 深交所：`0.` + 代码（如：`0.000651`）

**fields 参数值**
```
f12,f14,f2,f3,f15,f16,f18
```

**请求示例**
```
https://push2.eastmoney.com/api/qt/ulist.np/get?fltt=2&secids=1.510500,0.000651&fields=f12,f14,f2,f3,f15,f16,f18
```

**响应格式**

返回格式：JSON

**响应示例**
```json
{
  "data": {
    "diff": [
      {
        "f12": "510500",
        "f14": "500ETF",
        "f2": 7.224,
        "f3": 1.05,
        "f15": 7.280,
        "f16": 7.085,
        "f18": 7.149
      }
    ]
  }
}
```

**字段说明**

| 字段 | 说明           | 备注                     |
|------|----------------|--------------------------|
| f12  | 股票代码       |                          |
| f14  | 股票名称       |                          |
| f2   | 当前价格       | 可能不存在               |
| f3   | 涨跌幅（百分比）| 需除以100，如：1.05 表示 1.05% |
| f15  | 最高价         |                          |
| f16  | 最低价         |                          |
| f18  | 昨收价格       |                          |

**备用字段**
- `f43` - 当前价格（备用）
- `f44` - 最高价（备用）
- `f45` - 最低价（备用）
- `f60` - 昨收价格（备用）
- `f170` - 涨跌幅（备用）

**注意事项**
- 接口仅支持A股（上交所和深交所）
- 必须携带 `Referer` 请求头
- 涨跌幅已提供百分比，但需除以100
- 响应可能为 `{}`，需要做空值处理

### 3.2 搜索股票

**接口地址**
```
GET https://searchapi.eastmoney.com/api/suggest/get?input={keyword}&type=14&token={token}
```

**请求头**
```
Referer: https://quote.eastmoney.com/
Accept: application/json,text/plain,*/*
```

**请求参数**

| 参数名 | 类型   | 必填 | 说明                    |
|--------|--------|------|-------------------------|
| input  | string | 是   | 搜索关键词              |
| type   | string | 是   | 固定值 `14`             |
| token  | string | 是   | 固定值 `D43BF722C8E33BDC906FB84D85E326E8` |

**请求示例**
```
https://searchapi.eastmoney.com/api/suggest/get?input=贵州茅台&type=14&token=D43BF722C8E33BDC906FB84D85E326E8
```

**响应格式**

返回格式：JSON

**响应示例**
```json
{
  "QuotationCodeTable": {
    "Data": [
      {
        "Code": "600519",
        "Name": "贵州茅台",
        "MktNum": "1",
        "QuoteID": "1.600519"
      }
    ]
  }
}
```

**字段说明**

| 字段   | 说明       | 备注                     |
|--------|------------|--------------------------|
| Code   | 股票代码   |                          |
| Name   | 股票名称   |                          |
| MktNum | 市场编号   | 1-上交所，0-深交所       |
| QuoteID| 行情ID     | 格式：`{market}.{code}`  |

**市场编号映射**
- `1` → 上交所（前缀：SH）
- `0` → 深交所（前缀：SZ）

## 五、使用建议

### 5.1 数据源选择

1. **默认使用自动兜底模式**：`stocks.auto` 会按腾讯 → 新浪 → 东方财富的顺序自动切换
2. **需要港股/美股数据**：选择腾讯或新浪
3. **仅需A股数据**：可选择东方财富（JSON格式更易解析）
4. **高频调用场景**：建议缓存数据，避免频繁请求第三方接口

### 5.2 错误处理

1. **股票代码不存在**：各接口返回格式不同，需要根据返回值判断
2. **接口超时/失败**：使用自动兜底模式，自动切换备用数据源
3. **编码问题**：腾讯和新浪接口返回GBK/GB18030编码，需正确转换

### 5.3 频率限制

第三方接口通常有频率限制，建议：
- 在服务层实现短TTL缓存（如5-10秒）
- 批量查询使用 `getStocks` 而非多次调用 `getStock`
- 生产环境考虑限流措施
